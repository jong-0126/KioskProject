# Kiosk 프로젝트

자바를 이용하여 키오스크를 개발해 보았습니다. 다양한 기능을 추가하여 점차적으로 프로젝트를 완성하였습니다.

## 📅 개발 기간

- **2025-03-07 ~ 2025-03-14**

## 💻 개발 환경

- **Java**: 17
- **JDK**: 17.0.11
- **빌드 도구**: Gradle
- **IDE**: IntelliJ IDEA
- **버전 관리**: Git, GitHub

## 🚀 프로젝트 목표

이 프로젝트는 자바를 사용하여 키오스크 시스템을 개발하는 것입니다. 사용자는 메뉴를 선택하고, 장바구니에 추가하며, 주문을 완료하는 과정을 통해 실제 키오스크의 흐름을 구현합니다.

## 🛠️ 개발 과정

### Lv1: 기본적인 키오스크 기능 프로그래밍

- 메뉴 번호와 이름, 가격, 설명을 표시하고 번호를 입력받는 방식으로 메뉴 선택
- 메뉴가 선택되면 선택된 메뉴를 보여주고 프로그램 종료

### Lv2: 햄버거 메뉴를 클래스로 관리하기

- **MenuItem** 클래스를 생성하여 음식 항목을 관리
  - **MenuItem** 클래스는 이름, 가격, 설명을 필드로 갖는다.

### Lv3: Main 함수에서 관리하던 전체 순서 제어를 Kiosk 클래스를 통해 관리

### Lv4: Menu 클래스 생성하여 MenuItem 클래스를 관리

- 음식 메뉴와 주문을 나누어 개발

### Lv5: 캡슐화 적용

- 객체 지향 설계를 적용하여, 데이터와 메서드를 캡슐화하여 코드의 유지보수성을 높임

## 🎯 도전과제

### 장바구니 기능, 구매하기 기능, 할인율 계산 기능 추가

- **Lv1**: 장바구니 기능, 구매하기 기능
- **Lv2**: 할인율 기능
  - **enum**을 활용하여 할인율을 적용

## 📌 기능 설명

1. **메뉴 선택**: 사용자는 번호를 입력하여 원하는 메뉴를 선택합니다.
2. **장바구니 기능**: 선택한 메뉴는 장바구니에 추가되어 관리됩니다.
3. **구매하기**: 장바구니에 담긴 메뉴를 확인하고 주문을 완료할 수 있습니다.
4. **할인율 계산**: 할인율을 적용하여 최종 결제 금액을 계산합니다. 할인은 국가유공자, 군인, 학생 등 다양한 조건에 따라 달라집니다.
